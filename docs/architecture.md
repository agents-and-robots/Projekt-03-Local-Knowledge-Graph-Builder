# Architektur – Local Knowledge Graph Builder

## Module

- extractor.py – extrahiert Entities
- relationer.py – erkennt Relationen
- graph_builder.py – baut Knowledge Graph
- agent.py – orchestriert den Prozess

## Datenfluss

1. Extractor extrahiert Entities
2. Relationer erkennt Relationen
3. Graph Builder erzeugt Graph
4. Agent führt Feedback-Loop aus
